# 正規表達式
`/\B(?=(\d{3})+(?!\d))/g`將可匹配出每三位數字的空格
搭配`'12345678'.replace(/\B(?=(\d{3})+(?!\d))/g, ',');`使用可對字串中的數字加上千位符號：**12,345,678**

`\B`代表的是不匹配最右與最左的邊界（邊界為空字串）

`(?=(\d{3})+(?!\d))`是本範例的關鍵
需拆分為`(?=  )`與`(\d{3})+(?!\d)`兩個部分來看

`(?=  )`為匹配滿足`?=`後方條件的 **空字串**
這是因為在`(?=  )`前方並沒有輸入字串或條件的因素
例如：若為`4(?=  )`則會匹配到4本身

`(\d{3})+(?!\d)`則是條件
